# 📇 Business Card / Profile App

A professional personal profile application built with **Flutter**. It showcases contact information, social stats, and branding in a clean, modern UI.

## 🔥 Highlights
* **Custom UI Components:** Reusable widgets for rows, icons, and images.
* **Responsive Design:** Uses `SingleChildScrollView` to ensure the layout works on all screen sizes.
* **Modern Aesthetics:** Features custom styling, shadows, and rounded cards.
* **Clean Architecture:** Well-organized project structure (`components`, `screens`, `assets`).

## 🛠️ Built With
* Flutter & Dart.
* Custom Assets (Fonts/Images).
